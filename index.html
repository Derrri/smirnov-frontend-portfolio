проект «Адаптивная и мобильная вёрстка»

PerfectPixel вёрстка

Цели:

  • Кроссбраузерная вёрстка под iOS, Android, Windows и macOS
  • Вёрстка корректно работает в следующих браузерах: Chrome, Firefox, Safari, Opera, Edge, IE11, Mobile Safari, Mobile Chrome
  • Соответствие вёрстки макету по PerfectPixel
  • Промежуточные состояния для устройств с шириной экрана 360px, 768px и 1200px с помощью принципа «Резиновая вёрстка»

  • Семантическое использование тегов и названий атрибутов
